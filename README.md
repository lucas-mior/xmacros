# xmacros
Some C macro tricks

## Files
- `generic.c`: needed by the other files, handles the C primitives
- `assert.c`: defines type safe assertions for the primitives
- `xenums.c`: for creating enums with introspection
- `xstructs.c`: for creating structs with introspection
- `fmt_functions.c`: for working with introspected structs
- `main.c`: demonstration of the usage
